Old Sounds v1.20_1 (compatible with Minecraft versions 1.20.1+) by Sencie
https://www.planetminecraft.com/texture-pack/old-sounds-4911323/
‘Default’ version. The pack brings back the old sounds and no sounds were muted from the game (also includes splash texts from Beta 1.7.3).
To preserve the consistency of the game experience, some new sounds are changed to old style.

This pack only changes the sounds. Please use it with the Programmer Art pack 
or another retro texture pack such as Developer Art Classic https://www.planetminecraft.com/texture-pack/devart-classic/
